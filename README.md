# Sourcing Command

A full-stack-style front-end prototype for electronics component supplier award decisions. It models suppliers, demand hubs, routes, risks, product details, scoring confidence, workflow intelligence, document imports, and scenario dashboards.

Live URL: https://sourcing-command.vercel.app

## Run Locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
